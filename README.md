# Neskkarangal Charity Website

Starter structure for a charity website focused on community care, outreach, and volunteering.

## Project structure

- index.html — homepage landing page
- assets/css/styles.css — visual design and responsive layout
- assets/js/main.js — small interactive behavior for navigation
- assets/img/ — place logos, photos, and icons here
